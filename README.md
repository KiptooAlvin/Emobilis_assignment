#  Global Tech Summit 2026 — Event Landing Page

Welcome to the **Global Tech Summit 2026** project — a modern, responsive event landing page designed to showcase a global technology conference. This webpage highlights the event details, schedule, and call-to-action buttons for registration and agenda viewing.

---

##  Project Overview

This project is a clean and elegant **HTML & CSS-based** landing page created to represent the **Global Tech Summit 2026** — an event bringing together leaders, innovators, and visionaries in the tech industry.

The layout is fully responsive and optimized for readability, combining a clear text section with a compelling right-side hero image that emphasizes engagement and innovation.

---

##  Features

✅ Responsive design  
✅ Clean navigation bar  
✅ Balanced layout  
✅ Elegant typography and color contrast  
 

---

##  Technologies Used

- **HTML5** – Page structure and content  
- **CSS3 (Flexbox)** – Styling, responsiveness, and layout alignment
